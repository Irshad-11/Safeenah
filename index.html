THE PURPOSE
"Islam calls us to seek truth with integrity —
not to blindly follow traditions.
If the truth points somewhere,
that is exactly the direction
an honest mind should follow."

Most of what happened in early Islamic history has been shaped and filtered through centuries of interpretation and narrative framing. SAFEENAH exists to present these histories through original sources and references, so anyone can examine them directly without being limited to a single perspective.

Is This For You?

✓ THIS IS FOR YOU IF

  • ◆ You want to understand Islamic history — not a curated version of it
  • ◆ You can sit with evidence honestly, even when it's uncomfortable
  • ◆ You're tired of different answers from different groups and want primary sources
  • ◆ You understand that following truth matters more than defending your school of thought

✕ THIS IS NOT FOR YOU IF

  • ◆ You've already decided what the answer is and are not truly looking
  • ◆ Religious identity matters more to you than what the sources actually say
  • ◆ You want to use this as a tool to "win" an argument rather than find truth
  • ◆ You call others' research blind without doing the actual work yourself

UNDER THE HOOD

How It Works

Completely data-driven. No CMS. No server. Add a JSON file — content appears.

01

ADD JSON

Create a new .json file in the data/events/ folder with the event details and references

02

UPDATE INDEX

Add one line to data/events/index.json with the event ID, title, and year

03

DONE

The event appears in the timeline automatically. Its own permanent URL is live. No HTML editing required.

PERMANENT URL GENERATED

/Safeenah/event/?id=ghadir-khum
OPEN CONTRIBUTION

Help Build This

ONE RULE FOR CONTENT

Every claim needs a reference. Not a blog post, not a YouTube video. A primary source — or a recognized classical source — with volume and page number.

That's the only rule.
